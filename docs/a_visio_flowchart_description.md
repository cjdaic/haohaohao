# a.txt 对应流程图说明

## 流程主线

1. 开始
2. 导入工件模型（STL / OBJ）
3. 判断工件曲面是否简单
4. 简单曲面：二面角聚类（设阈值）提取 Patch
5. 复杂曲面：手动设置加工平面并提取待加工区域
6. 汇合为待加工曲面 Patch
7. 参数化展开到 UV 平面
8. 选择参数化算法（LSCM / ARAP）
9. 在 UV 面片应用并编辑 SVG 纹理
10. 映射回曲面并通过 Alpha 裁剪获得路径边界
11. 设置加工参数与图案参数（间隔、角度、策略）
12. 生成路径（往返、回型、轮廓、等 Z、弧线填充）
13. 在 3D / UV / 路径预览中联合检查结果
14. 判断结果是否确认无误
15. 否：调整参数或策略后回到“设置参数/生成路径”环节
16. 是：导出 CSV/任务文件或直接下发执行
17. 结束

## 关键分支

- 分支 A：`工件曲面是否简单？`
  - 是：走二面角聚类提取
  - 否：走手动平面提取
- 分支 B：`选择参数化算法`
  - LSCM：保角优先，速度快
  - ARAP：局部刚性优先，畸变更均匀
- 分支 C：`结果是否确认无误？`
  - 否：回环迭代调整
  - 是：导出或执行
