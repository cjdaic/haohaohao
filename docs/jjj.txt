软件应用流程说明（修订版）

本软件面向曲面激光加工场景，整体流程为“模型导入 → 待加工面片确定 → 参数化展开 → SVG图案贴附与编辑 → 路径生成与预览 → 导出/执行”。

第一步：导入工件模型并确定待加工面片。
在左上角导入待加工工件的 STL 或 OBJ 文件。对结构较规整的模型，可先执行二面角聚类：系统根据相邻三角面二面角与阈值将网格自动分组为多个 patch，用户选择目标 patch 作为加工区域。对复杂曲面，可通过调整聚类阈值并结合手动选中 patch 的方式确定加工面片，保证加工区域尽可能完整且可控。

第二步：对选中面片进行参数化（3D→UV）。
系统提供 LSCM 与 ARAP 两种参数化方案。LSCM 偏重保角，角度畸变较小、计算速度较快；ARAP 偏重局部刚性保持，整体形状变形更均匀，适合对形状保真要求更高的场景。用户可根据曲面特征与工艺目标选择方案。

第三步：应用 SVG 图案并编辑纹理变换。
在目标 patch 上应用 SVG 图案后，可在纹理编辑器中对图案进行缩放、平移和旋转。系统会在参数域中完成图案映射，并在回到模型曲面时保持与当前贴图变换一致；同时基于图案 Alpha 区域进行有效加工范围约束，为后续路径裁剪提供精确边界。

第四步：设置加工参数并生成路径。
在右侧参数区设置工艺与图案参数（如扫描间隔、扫描角度、方向模式、畸变控制等），并选择路径策略。当前支持的策略包括：轴向往返填充、圆弧往返填充、回型填充（内→外/外→内）以及 Z 轴分层填充。系统先在 UV 域生成候选路径，再裁剪到 patch 与 SVG 覆盖区域，最后映射回三维曲面生成加工轨迹。

路径规划算法细节（当前实现）
1. 轴向往返填充（line_hatch）
先将边界按设定角度旋转到局部坐标系，再按固定间距做平行扫描线；每条扫描线与多边形边界求交，交点成对配对得到有效加工线段。相邻线段采用“蛇形往返”连接（偶数条正向、奇数条反向），减少空移距离。

2. 圆弧往返填充（arc_hatch）
该策略采用“空间曲面弧线扫描”方式，直接在圆弧曲面上生成三维轨迹（不是先平面填充再整体投影）。核心流程如下：

（1）圆弧几何参数确定
先读取用户输入的圆心与半径；再基于当前 patch 的网格点进行圆弧拟合与一致性检查。若输入参数缺失、偏差过大或不稳定，系统自动回退到拟合值，得到稳定的 `center_x / center_axis2 / radius`。

（2）扫描坐标系建立
根据 patch 法向自动判定“行方向轴”（在实现中会在 `y` 或 `z` 方向二选一），将圆弧曲面参数化为：
- 弧向参数：`theta`（沿圆弧展开方向）
- 行向参数：`row`（逐行推进方向）
随后统计该 patch 的有效 `theta` 范围和 `row` 范围，作为扫描窗口，避免越界扫描。

（3）按行推进（spacing 控制）
按用户设定的 `spacing` 在 `row` 方向离散出多行扫描线；每一行对应一个固定 `row_value`。这一步决定道间距与覆盖密度，是加工均匀性的主要控制参数。

（4）行内弧向采样与遮罩判定
在每一行内，对 `theta` 做高密度采样（采样步长与 spacing 相关，并限制上下界，兼顾精度和速度）。每个采样点会映射到归一化参数 `(u_norm, v_norm)`，再查询 SVG Alpha 遮罩，判断该点是否落在图案有效区域。

（5）有效区间提取（区间化而非逐点直连）
对“在遮罩内/外”的布尔序列做边界检测，提取连续的有效 `theta` 区间 `[theta_start, theta_end]`。这一步把离散采样转成连续弧段，避免路径出现锯齿化的开关光抖动。

（6）往返扫描（蛇形）组织
相邻行使用蛇形策略：奇偶行交替反向，减少行间空移。对每个有效区间按当前行方向确定起终点，形成连续的“去程-回程”加工模式。

（7）弧段速度一致性重采样
对每个有效弧段按“目标扫描速度 + 控制周期（10us）”计算采样点数，进行弧长一致重采样，确保点间距与速度匹配，便于后续执行端稳定跟踪。

（8）三维点直接生成与断点标记
每个重采样点直接按圆弧方程生成 XYZ：
- `x = center_x + radius * sin(theta)`
- 另一圆弧轴坐标为 `center_axis2 + radius * cos(theta)`
- 行方向坐标为 `row_value`
新弧段起点自动标记 `jump_before`，段末标记箭头点，用于预览与后续 MARK/JUMP 分段生成。

（9）结果特性
该算法对圆弧类曲面更稳定，优点是：几何贴合好、行间覆盖可控、与 SVG 覆盖区一致；同时通过蛇形与重采样控制，有效降低空移与速度波动。

（10）论文写法参考（可直接改写进方法章节）
1) 变量定义
- `C = (c_x, c_a)`：圆弧中心（`a` 表示非 `x` 的圆弧轴，具体为 `y` 或 `z`）
- `R`：圆弧半径
- `theta in [theta_min, theta_max]`：弧向参数
- `r in [r_min, r_max]`：行向参数（逐行推进）
- `Delta_r`：行间距（对应 `spacing`）
- `v_scan`：扫描速度（mm/s）
- `T_ctrl = 10 us`：控制周期
- `A(u,v)`：SVG Alpha 遮罩函数，阈值 `tau`（实现中约为 `16/255`）

2) 曲面参数化模型
将圆弧曲面上的加工点写成二参数形式 `P(theta, r)`：
- `x(theta, r) = c_x + R * sin(theta)`
- `a(theta, r) = c_a + R * cos(theta)`
- `b(theta, r) = r`
其中 `(a,b)` 在实际实现中由 patch 法向自动决定为 `(y,z)` 或 `(z,y)` 的排列。

归一化映射用于遮罩查询：
- `u_n = (theta - theta_min) / (theta_max - theta_min)`
- `v_n = (r - r_min) / (r_max - r_min)`
当 `A(u_n, v_n) > tau` 时判定该点属于有效加工区域。

3) 伪代码（空间曲面弧线扫描）
```
Input: patch mesh M, SVG mask A, spacing Delta_r, speed v_scan
Output: 3D path points P and break flags B

Estimate/repair (C, R) from patch geometry
Compute valid ranges [theta_min, theta_max], [r_min, r_max]

N_row = floor((r_max - r_min)/Delta_r) + 1
for i = 0 .. N_row-1:
    r_i = min(r_min + i * Delta_r, r_max)
    Sample theta densely on [theta_min, theta_max]
    Build inside/outside boolean sequence by A(u_n, v_n)
    Extract contiguous valid theta intervals I_i = {[theta_s, theta_e]}

    if serpentine direction is reverse:
        reverse interval order and direction

    for each interval [theta_s, theta_e] in I_i:
        L = R * abs(theta_e - theta_s)                 # arc length
        N_seg = clamp(floor(L / (v_scan * T_ctrl)) + 1, 2, 50000)
        for k = 0 .. N_seg-1:
            t = k / (N_seg-1)
            theta_k = theta_s + t * (theta_e - theta_s)
            generate 3D point P(theta_k, r_i)
            append point and flags (jump_before at segment start)
toggle serpentine direction
```

4) 复杂度与实现约束
- 主体复杂度约为 `O(N_row * N_theta + N_out)`，其中 `N_out` 为输出点数。
- 当前实现中：
  - 行数上限约束：`N_row <= 200000`
  - 弧向采样数：`N_theta` 限制在 `[256, 20000]`
  - 行内采样步长：`step_mask = clamp(0.5*spacing, 0.002, 0.05) mm`
  - 弧段重采样点数：`N_seg in [2, 50000]`
这些约束用于在精度、稳定性和计算时间之间取得平衡。

5) 可用于论文的算法优势表述
该方法通过“曲面原位建模 + 遮罩区间化 + 速度一致重采样”替代传统“平面填充后整体投影”，可显著降低圆弧区域的轨迹漂移与边界抖动；同时，蛇形扫描减少空移距离，区间化裁剪减少无效开关光，整体上提高了曲面图案加工的一致性与执行稳定性。

3. 回型填充（ring，内→外 / 外→内）
以边界包围盒中心为基准，按间距生成一组同心矩形环。内→外模式从小环逐步扩展，外→内模式反向收缩。每一环闭合输出，层与层之间设置断点，便于后续生成跳转段。

4. Z 轴分层填充（z_layer）
按层高对选中面片做多层 Z 截面。每一层通过“网格三角形与 Z 平面求交”得到线段，再将线段自动串接为连续链，并做局部平滑处理。层间与链间自动加断点，随后可再按 SVG 遮罩二次裁剪，仅保留图案覆盖区域。

5. 公共后处理逻辑
所有策略生成的候选路径都会经过统一处理：先裁剪到 patch 有效域，再按 SVG Alpha 遮罩精确裁剪，然后进行 UV→XYZ 映射（基于三角形定位与重心插值），最后转换为 MARK/JUMP 分段并执行拐角减速等后处理。

第五步：预览、导出与执行。
生成后可在 3D 视图、UV 视图和路径预览窗口联合检查结果（含分段、开关光与路径方向）。确认无误后可导出路径 CSV/任务文件，或直接执行任务下发。

说明：
当前版本的“实时性”主要体现在用户调整参数、纹理或策略后触发快速重算与刷新预览，而非后台持续流式求解。
