# 曲面变参数加工数学模型

版本基线：2026-05-06  
适用范围：当前激光器接线支持变功率、变速度，不假设可动态变频、变焦或实时改光斑直径。  
参考资料：`reference/二维振镜超快激光高精加工球面等角螺旋槽研究_中国激光.pdf`

## 1. 目标

曲面激光加工中，同一套平面参数在高曲率区域容易出现以下问题：

- 单位面积能量密度不均。
- 脉冲沿扫描方向的搭接率变化。
- 相邻扫描道在曲面上的实际间距变化。
- 光斑投影变形导致局部加工宽度、深度或粗糙度变化。

当前硬件能动态控制的量是：

- 激光功率 `P`。
- 扫描速度 `v`。

因此建模目标是：在频率 `f` 和名义光斑直径 `d` 固定时，通过 `P(s)` 与 `v(s)` 补偿不同曲率位置的光斑搭接率和单位面积能量输入，使加工尽量平滑均匀。

## 2. 基础搭接率模型

### 2.1 脉冲方向搭接率

论文中使用的扫描方向脉冲重叠率可写为：

```text
eta_p = 1 - v / (d * f)
```

其中：

- `eta_p`：同一条扫描线上相邻脉冲的搭接率。
- `v`：扫描速度，单位 mm/s。
- `d`：加工表面上的等效光斑直径，单位 mm。
- `f`：脉冲重复频率，单位 Hz。

反解目标速度：

```text
v_target = d_eff * f * (1 - eta_p_target)
```

当前接线可控速度，所以保持目标脉冲重叠率的首选控制量是 `v`。

### 2.2 道间搭接率

相邻扫描道间的搭接率可近似为：

```text
eta_l = 1 - h / d_eff
```

其中：

- `eta_l`：相邻扫描道之间的搭接率。
- `h`：曲面上的实际道间距，单位 mm。
- `d_eff`：曲面上的等效光斑直径，单位 mm。

若路径是在 UV 或平面域生成，`h` 应尽量用曲面距离估计，而不是只看 UV 距离。

### 2.3 单位长度与单位面积能量

单位扫描长度能量：

```text
E_l = P / v
```

单位面积能量的近似：

```text
E_a = P / (v * h)
```

当路径间距 `h` 固定而曲率使有效光斑或局部曲面面积变化时，可通过调整 `P/v` 保持能量输入稳定。

## 3. 曲率对光斑和搭接率的影响

### 3.1 曲率半径

对于曲面上一点，取主曲率：

```text
k1, k2
```

当前工程实现使用法向变化估计一个曲率代理值：

```text
kappa_i = mean_j( acos(n_i dot n_j) / |x_i - x_j| )
```

其中 `j` 为顶点 `i` 的邻接顶点。它不是严格主曲率，但能反映局部弯曲程度，适合做工艺参数渐变的驱动量。

归一化：

```text
k_norm = clamp(kappa / percentile95(kappa), 0, 1)
```

### 3.2 曲率导致的等效光斑变化

严格模型需要考虑：

- 入射方向与曲面法向夹角 `theta`。
- 离焦量 `Delta z`。
- 曲率半径 `R = 1 / |kappa|`。
- 振镜扫描导致的局部投影变形。

通用近似：

```text
d_eff = d0 * g(theta, Delta z, kappa)
```

入射角主导时可用：

```text
d_eff_major = d0 / cos(theta)
d_eff_minor = d0
d_eff_area_factor = 1 / cos(theta)
```

离焦主导时可用高斯光束近似：

```text
w(z) = w0 * sqrt(1 + (Delta z / z_R)^2)
d_eff = 2 * w(z)
```

当前软件暂未稳定维护每个路径点的入射角和离焦量，所以第一版采用曲率代理进行保守补偿：

```text
v(s) = clamp(v0 * (1 - alpha_v * k_norm), v_min, v_max)
P(s) = clamp(P_base(k_norm) * sqrt(v0 / v(s)), P_min, P_max)
```

其中：

- `alpha_v`：高曲率降速强度，当前实现为 `0.45`。
- `sqrt(v0 / v)`：在降速时避免单位长度能量过高或过低的温和补偿项。

## 4. 当前工程参数

当前 `CurvatureModel` 默认值：

```text
spot_diameter_mm = 0.03
pulse_frequency_hz = 20000
target_pulse_overlap = 0.70
target_line_overlap = 0.50
power_range = 10 W 到 30 W
speed_range = 200 mm/s 到 500 mm/s
```

名义速度：

```text
v0 = d * f * (1 - eta_p)
   = 0.03 * 20000 * (1 - 0.70)
   = 180 mm/s
```

因为当前最小速度默认是 `200 mm/s`，实际会被夹到 `200 mm/s`。如果希望严格达到 70% 脉冲重叠率，应把最小速度调低或提高光斑直径/频率。

## 5. 推荐调参流程

1. 标定实际光斑直径 `d0`。
2. 确认激光频率 `f`，当前接线若不能动态变频，则固定使用激光器设置中的频率。
3. 选择目标扫描方向搭接率 `eta_p_target`，精加工可从 `0.65-0.85` 试起。
4. 选择目标道间搭接率 `eta_l_target`，可从 `0.40-0.70` 试起。
5. 按公式计算名义速度 `v0`。
6. 设定速度范围，确保 `v0` 落在范围内。
7. 在平面样块上标定 `P_base` 与 `v0`。
8. 在不同曲率样块上逐步增加 `alpha_v` 和功率上限。
9. 通过槽宽、槽深、粗糙度和烧蚀边缘评估均匀性。

## 6. 后续可升级模型

若后续能获得稳定的路径点法向和光轴方向，可升级为入射角模型：

```text
theta = acos(|n dot b|)
d_eff = d0 / sqrt(cos(theta))
v = clamp(d_eff * f * (1 - eta_p_target), v_min, v_max)
P = clamp(P0 * d_eff / d0 * v / v0, P_min, P_max)
```

其中：

- `n`：曲面单位法向。
- `b`：光束单位方向。
- `d_eff / d0`：补偿光斑面积变化。
- `v / v0`：补偿单位长度能量变化。

若后续能动态变焦或使用五轴姿态保持法向入射，则曲率补偿可以减弱，主要保留道间距和局部面积补偿。
